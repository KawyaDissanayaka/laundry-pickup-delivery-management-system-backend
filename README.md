A Spring Boot backend API for managing laundry pickups, deliveries, orders, customers, and drivers. Provides REST endpoints for CRUD operations and integrates with a database for persistent storage.

- **View on GitHub:** [GitHub Repo Link](https://github.com/KawyaDissanayaka/laundry-pickup-system-backend)

---

## Technology Stack

- **Framework:** Spring Boot 3.x  
- **Language:** Java 17+  
- **Database:** H2 (development) / MySQL or PostgreSQL (production)  
- **Build Tool:** Maven  
- **API Type:** RESTful API  

---

## Features

- Manage customers, drivers, and laundry orders  
- CRUD operations for all entities  
- Dashboard endpoints for analytics  
- H2 database for quick development  
- Option to switch to MySQL/PostgreSQL for production  
- Exception handling and input validation  

---

## Prerequisites

- Java 17+ installed  
- Maven installed  
- Optional: MySQL or PostgreSQL if using a production database
